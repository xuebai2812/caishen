# 谈判策略与话术

## 一、先理解双方诉求（第一性原理）

公司裁员时真正想要的，不是"省那点补偿金"，而是：**快速、干净、低风险地完成减员**——最好让你"自愿"签字走人，避免走裁员法定程序、避免仲裁、避免舆情。

你的诉求通常是（按本项目目标排序）：**拿到最大补偿、尽量延长在职时间、体面收尾不留 gap**。

谈判空间就藏在这两组诉求的差里：公司愿意为"快和干净"付溢价，你可以用"配合签字、保密、不闹"去交换更高金额或更长在职期。

## 二、关键原则

1. **没算清、没想好之前，什么都别签、别录口头承诺。** 尤其别签任何含"自愿离职""个人原因""协商一致且无其他争议"字样的文件。
2. **区分法律底线和谈判筹码。** N 是单位提出协商解除的法律底线；2N 是违法解除时的杠杆。谈判是在"底线"之上争取,不要把可争取的当成应得的去赌气。
3. **让公司先出价。** 先问"公司方案是什么"，不要自己先报数字。
4. **保持体面、不撕破脸，但明确传递"我懂规则"。** 让 HR 意识到你知道 N/2N、知道仲裁流程，谈判地位立刻不同——多数低价方案是赌你不懂。
5. **拖延本身是筹码。** 你在职一天，公司的减员就没完成一天；只要不签字，时间站在你这边（注意时效另算）。

## 三、HR 常见套路与反制

| HR 套路 | 真相 | 反制 |
|---|---|---|
| "公司政策就是 N+1，这是上限" | N+1 是协商话术，不是法定上限；违法解除可主张 2N | "我先了解下解除的法律依据，再谈具体数字" |
| "你绩效不达标，属于不能胜任" | 不能胜任需有考核标准+证据，且须先培训/调岗，否则违法 | 不在绩效结论上签字认可；要求出示考核依据 |
| "今天不签，明天方案就没了/更低" | 制造紧迫感逼你就范 | "重大决定我需要时间考虑，急不来" |
| "走自愿离职流程，对你找下家好看" | 自愿离职=放弃全部补偿金 | 坚决不写"个人原因/自愿" |
| "先把工作交接了，补偿随后到账" | 失去筹码后易被拖延/变卦 | 补偿到账或协议白纸黑字签好再交接 |
| 口头承诺加钱 | 口头无凭 | 一切写进书面协议 |

## 四、三类目标的具体话术

### 抬价（争取更高补偿）
- "我对继续在这边工作是有意愿的，如果公司希望我配合协商离开，那补偿方案需要让我能接受。"
- 摆事实而非情绪："我在这干了 X 年，手上还有 XX 项目，按 N 我能拿到 A，如果是协商解除我希望是 N+X。"
- 如有违法解除迹象："这次解除的法律依据我还没看明白，按我了解的情况可能构成违法解除，那对应的是 2N。我们能不能在这中间找个双方都能接受的数？"（把 2N 作为锚点，落在 N 和 2N 之间）
- **"继续履行"反将一军**（公司最怕你回来）："如果按违法解除算，我其实可以不要赔偿、直接主张继续履行劳动合同回来上班，期间工资公司也得补。我个人更倾向于好聚好散，但前提是补偿得谈到位。"——用"我可以回来"逼出更高的"让我走"的价。

### 延长在职时间（符合本项目目标）
- "手上的 XX 项目正在关键节点，我离开会有交接风险，建议让我做完这个阶段，在职到 X 月。"
- 用"平稳交接/降低公司风险"为理由，争取多发 1-2 个月工资和社保连续。
- 在职期社保不断 = 避免 gap、利于下家背调和贷款。

### 体面收尾、不留 gap
- 要求开具**离职证明**（第 50 条，单位法定义务），且不得写负面评价。
- 协商离职日期与下家入职衔接，争取"带薪找工作"的缓冲。
- 确认年终奖、未休年假折现、报销等一并结清。

## 五、谈判节奏建议

1. 第一次谈：只听不签，问清方案和解除依据，表达"需要时间考虑"。
2. 回去算钱（模块一）、判断是否有 2N 杠杆、定好自己的目标价和底价。
3. 第二次谈：用事实和锚点抬价，把口头变书面。
4. 谈拢：逐字看协议（重点看金额、支付时间、是否含"放弃所有权利"条款、离职证明），无误再签。
5. 谈不拢：保留证据，走仲裁（模块三）。仲裁不是撕破脸，是正常维权，也常促使公司回到谈判桌。
