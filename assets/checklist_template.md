## 二、签字前红线（每项打勾确认）

- [ ] 没有签任何含"自愿离职""个人原因"字样的文件
- [ ] 没有在"绩效不合格 / 不能胜任"确认书上签字认可（如签，已注明"已收到，有异议"）
- [ ] 协议里的金额、是否税前、支付时间都已核对清楚
- [ ] 识别并处理了"已结清全部款项 / 放弃一切争议"这类一揽子放弃条款
- [ ] 确认了竞业限制条款是否启用、补偿多少、期限多长
- [ ] 要求公司开具《离职证明》，且无负面评价
- [ ] 口头承诺全部落到了书面协议

## 四、劳动仲裁材料清单（谈不拢时用）

- [ ] 劳动仲裁申请书（本人 1 份 + 仲裁委 1 份 + 按被申请人数各 1 份）

可一并主张（别漏）：□ 未休年假折现（日工资 3 倍）□ 加班费 □ 欠发奖金/提成/报销 □ 未签合同二倍工资

## 五、我的行动顺序

1. 第一次谈：只听不签，要到写明理由的解除通知，说"需要时间考虑"。
2. 回来算清账、判断有没有 2N / 继续履行杠杆、定目标价和底价。
3. 第二次谈：用"配合签字 + 平稳交接"换"更高补偿 + 延长在职 + 干净离职证明"，落到书面再签。
4. 谈不拢：保留证据，1 年内申请劳动仲裁（免费）。

> 喜迎裁神——被裁不是失败，是带着一笔钱体面离场、换一个更好开始的机会。把牌打好。
