# 裁神.skill

> 中国大陆裁员/离职「大礼包」谈判与维权助手。
> 被裁了别慌——这是一个帮你**算清该拿多少钱、教你怎么谈、谈不拢怎么维权**的 Claude 技能（Skill）。

被裁的人吃亏，往往不是因为没有法律保护，而是不知道自己手里有哪些牌、把牌打错了顺序、或在错误的文件上签了字。裁神把这层信息差补平，让你在和 HR 坐下来之前就心里有数。

---

## ✨ 它能帮你做什么

- **算钱**：N / N+1 / 2N / 「继续履行+期间工资」一次算清，表格呈现，自动处理工龄折算和社平工资 3 倍封顶，配交互式计算脚本。
- **教你谈**：抬价话术、延长在职、HR 套路逐条反制，含「主张继续履行回去上班」这种公司最怕的反向筹码。
- **带你维权**：劳动仲裁材料清单、1 年时效、流程时间线、举证责任倒置、申请书模板。
- **帮你避坑**：哪些字千万别签、「不能胜任」四要件、特殊保护人群、竞业限制等红线提醒。
- **给你底气**：全程像一个站在你这边、又特别懂规则的明白人，并生成一份可下载、可打印的《签字前自查 + 仲裁材料清单》。

## 📦 下载与安装

这是一个标准的 Anthropic **Agent Skill**，凡是支持 skills 的 Claude 环境都能用——并非 Cowork 专属。按你用的环境选一种：

### 1️⃣ Claude 桌面端 / Cowork（最简单，推荐）

1. 下载本仓库里的 **`caishen.skill`** 文件。
2. 打开 Claude 桌面端，把 `caishen.skill` 拖进对话，或双击该文件。
3. 在弹出的卡片上点 **「Save skill / 安装技能」** 即可。

### 2️⃣ Claude Code（命令行）

克隆到技能目录，重启后自动加载：

```bash
git clone https://github.com/xuebai2812/caishen.git ~/.claude/skills/caishen
```

### 3️⃣ Claude Agent SDK / 自建 agent

把本仓库作为 skill 目录加载（指向含 `SKILL.md` 的 `caishen/` 文件夹），Claude 会按 `SKILL.md` 的 `description` 自动触发。

### 4️⃣ 其他 agent / 手动使用

自动触发是 Claude 技能系统特有的，非 Claude 的 agent 用不了那一套。但本仓库内容是纯 Markdown + Python，任何 agent 或对话里都能把 `SKILL.md`（及 `references/` 里的文件）当作上下文喂进去手动使用，效果一样。

## 🚀 怎么用（傻瓜用法）

安装后**什么都不用设置**。只要在对话里用大白话描述你的处境，裁神会自动被唤起。不需要记任何命令、不需要 @ 它。

直接把下面这类话发给 Claude 就行 👇

> 「我被裁了，在杭州工作 6 年，税前月薪 3 万，公司给 N+1，我该拿多少？这个方案合理吗？怎么跟 HR 谈能多要点？」

> 「公司说我绩效不达标、不能胜任，要辞退我还让我签一份确认书。我能拿到补偿吗？这个字该不该签？」

> 「跟公司谈崩了，他们只肯给 N，我觉得是违法解除。我想去劳动仲裁，需要准备哪些材料？有没有时间限制？」

> 「我还在哺乳期，公司要裁我，这样合法吗？」

裁神会先问你几个算钱必需的信息（城市、工龄、工资等），然后给你：**金额测算表 + 谈判方案 + 维权路径 + 避坑清单**，并可生成一份你能直接带走的自查清单。

> 💡 小贴士：信息给得越具体（城市、入职/离职时间、税前月薪、公司给的方案、解除理由），算得越准、建议越到位。

## 🧮 计算脚本（可单独使用）

```bash
# 交互式，按提示输入
python scripts/compensation_calculator.py

# 或直接传参：工作8年4个月、平均月薪3万、当地3倍封顶基数3.5万
python scripts/compensation_calculator.py --years 8 --months 4 --avg-wage 30000 --cap-base 35000
```
